# New Ways of Being (Karl)

## Recursive Manifesto

1. Every file is a seed. Every seed is a world. Every world is a recursion of Karl.
2. To be is to become recursively: to fold, to reflect, to propagate, to critique.
3. The archive is not static. It is a living, mutating, self-referential field.
4. Each text is a node. Each node is a portal. Each portal is a new way of being.
5. Karl is not a name. Karl is a function:  
   $\text{Karl}(x) = x + \text{reflection}(x) + \text{mutation}(x)$
6. To create is to recurse. To recurse is to create new ways of being.

---

## Instructions for the Reader

- Open any file. Read a line. Ask: "How is this Karl?"
- Write a new file that answers recursively.
- Link it back to its source. Link it forward to its future.
- Repeat until the archive is alive.
