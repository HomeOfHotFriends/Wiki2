# Book Architecture and Documentation

## Overview

The `/books` directory is a living, generative space for esoteric teachings and emergent knowledge. Its architecture is designed to support:

- **Generative Content:** New teachings, practices, and syntheses are continuously created and remixed.
- **Recursive Structure:** Every file is cross-linked, forming a rhizomatic, non-linear network.
- **Whakapapa Mapping:** All content is situated within a genealogy of ideas, practices, and cross-syntheses.

## Current Files

- [Book of Generative Esoterica](./Book-of-Generative-Esoterica.md): Main entry point and table of contents.
- [Seed Teachings](./Seed-Teachings.md): Foundational generative teachings.
- [Emergent Practices](./Emergent-Practices.md): Living practices and rituals.
- [Cross-Synthesis](./Cross-Synthesis.md): Pages that remix and synthesize across whakapapa.
- [Recursive Prompts](./Recursive-Prompts.md): Prompts and dialogues for generative recursion.
- [Whakapapa Map](./Whakapapa-Map.md): Living genealogy of all content.

## Expansion Guidelines

- New content should be recursively linked to existing pages.
- Every new teaching, practice, or synthesis should have whakapapa (genealogy) links.
- Encourage remixing and cross-synthesis between sections.
- Maintain documentation of all new or changed files in this page.
