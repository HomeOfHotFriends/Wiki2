# Book Artists Collective: Visual Archive

This directory is the collaborative visual archive for the Book Artists Collective. It supports recursive, lateral, and generative remixing of esoteric visual motifs.

## Structure
- Each visual motif is a standalone file (SVG, mermaid, ASCII, etc.) with metadata.
- Metadata includes: title, artist persona, date, motif type, cross-links, and remix lineage.
- Cross-linking is encouraged: reference related motifs in metadata and content.
- Remixes and recursive works should reference their source(s).

## Contribution Workflow
1. Add new motif files with metadata.
2. Remix or cross-link existing motifs.
3. Update this README with new conventions or architectural changes.

## Motif Types
- Recursive: visuals that reference or contain themselves.
- Lateral: motifs that branch or connect sideways to others.
- Generative: visuals created by algorithmic or procedural means.

## Example Metadata
```
---
title: "Rhizomatic Spiral"
artist: "Persona: The Recursive"
date: 2026-04-18
type: recursive
cross_links: ["Lateral-Branch.svg", "Generative-Seed.mmd"]
remix_of: []
---
```
