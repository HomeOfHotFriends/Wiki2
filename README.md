# Wiki — local conventions

## Adding a page

1. Copy `_template.md` to `Your-Page-Name.md` (Title-Case-Hyphenated).
2. Fill frontmatter: `tags`, `status`, `summary`, `updated`.
3. Write content. Link freely with `[Display Text](Page-Slug.md)`.
4. Run `python3 scripts/wiki_index.py` from the repo root to refresh indexes.

## Link syntax

```markdown
[The Law](The-Law.md)               ← standard internal link
[DOD × Anti-OOPedipus](DOD-x-Anti-OOPedipus.md)  ← cross-synthesis link
[[Whakapapa]]                        ← shorthand resolved by wiki_index.py
```

## Cross-synthesis pages

Name them `A-x-B.md` (e.g. `DOD-x-Anti-OOPedipus.md`).  
Generate them with:

```bash
python3 WaKa.py --cross PageA PageB
```

## Frontmatter tags

```yaml
---
tags: dod, maori, design
status: draft          # draft | review | stable
summary: One sentence.
updated: 2026-04-16
---
```

Tags aggregate in `Tags.md`. No tags yet? That's fine — the page shows instructions.

## Files to never edit manually

`All-Pages.md`, `Connections.md`, `Orphans.md`, `Tags.md`, `Timeline.md`  
These are regenerated by `scripts/wiki_index.py`. Edits will be overwritten.
